# ⭐ Why this repository?

Finding trustworthy Windows utilities can be difficult.

There are hundreds of "debloaters", "gaming optimizers", and tweak scripts online.

Some are excellent.

Some are outdated.

Some are unsafe.

This repository aims to collect only trusted and well-known Windows utilities in one place.
